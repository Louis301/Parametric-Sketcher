# Parametric-Sketcher
GUI-интерфейс разбит на две части: левую и правую. В левой части располагается эпюр Монжа в трёх проекциях: вид спереди, сбоку, сверху. На эпюре можно указывать точки на смежных видах, задавая таким образом характерные точки примитивов типа ТОЧКА, ОТРЕЗОК. Множество созданных через эпюр объектов формируют модель, которая отрисовывается в правой части GUI.
